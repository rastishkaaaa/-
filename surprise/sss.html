C++

#include—включение заголовочных файлов в программу
<iostream>—подключает стандартный библиотечный заголовочный файл для работы с потоками ввода-вывода
using namespace std—использывание пространство имён std
int—тип данных
main—возвращение результата своего выполнения и запуск прорграммы
cout—вывод данных на экран, в данном случае "Fuck you!"
endl—перемещение на новую строку, например как в html <br>
return 0—указание на успешное завершение программы
